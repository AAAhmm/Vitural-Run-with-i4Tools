# VirtualRun - 爱思助手跑道虚拟定位

基于爱思助手（i4Tools）的虚拟定位自动化工具，生成逼真的田径跑道顺时针跑步轨迹，用于模拟 GPS 移动。

## 核心功能

- **跑道轨迹生成**：根据 6 个校准点（四角 + 上下弧顶）拟合标准田径场椭圆形路径，支持多圈连续运行
- **真实运动模拟**：加入横向摆动（wobble，模仿真人跑步左右偏移）和配速微变（pace variation），避免机械直线轨迹
- **全自动输入**：通过 Win32 底层 API（`keybd_event` / `mouse_event`）绕过 UIPI 限制，自动在爱思助手窗口内完成点击→粘贴→确认操作
- **独立单文件**：C# 启动器将 PowerShell 脚本嵌入资源，编译为单个 EXE，无需额外文件即可运行
- **自适应窗口缩放**：校准数据按比例适配窗口大小变化
- **ESC 随时停止**：运行中按 ESC 中断


每个校区的跑道通过 6 个 GPS 坐标点标定：四角（BR/TR/TL/BL）+ 上下弧形顶点。

## 自定义跑道（v3.1 新增）

将自定义坐标 `.txt` 文件放入 `tracks/` 文件夹，程序启动时会自动加载，校区选择菜单中会显示自定义条目。

### 文件格式

与 `location.txt` 格式一致，每文件定义一个跑道：

```
跑道名称：
地址：可选地址行
右下（纬度,经度）
右上（纬度,经度）
左上（纬度,经度）
左下（纬度,经度）
上顶点（纬度,经度）
下顶点（纬度,经度）
```

- 坐标用英文逗号 `,` 或中文逗号 `，` 分隔均可
- `地址：` 行为可选
- `#` 开头的行为注释，会被忽略
- 空行自动跳过
- 文件名即作为默认跑道名称（如文件内容有名称行则优先使用）

### 示例

`tracks/` 目录下已包含 `模板-示例操场.txt`，可参考其格式创建新文件。

## 使用环境

- **操作系统**：Windows 10 / 11
- **依赖**：爱思助手（i4Tools）已安装并运行
- **权限**：需要管理员权限（程序自动触发 UAC 提权）
- **.NET**：需要 .NET Framework 4.x（系统自带）

## 快速开始

### 1. 构建（仅首次）

双击 `build.ps1` 或执行：

```powershell
powershell -ExecutionPolicy Bypass -File build.ps1
```

生成 `VirtualRun.exe`（约 38 KB）。

### 2. 运行

1. 先启动爱思助手，打开「虚拟定位」功能页面
2. 双击 `VirtualRun.exe`，同意 UAC 提权
3. 首次使用需要**校准**：按提示将鼠标移到爱思助手窗口的对应控件上，按 Enter 记录位置
   - 纬度输入框
   - 经度输入框
   - 修改虚拟定位按钮
   - 弹窗确认按钮
4. 选择校区（内置校区或自定义跑道）
5. 输入目标距离（公里）
6. 程序自动开始运行，控制台最小化，爱思助手保持前台

### 3. 诊断（出问题时）

```powershell
powershell -ExecutionPolicy Bypass -File diag.ps1
```

逐步测试剪贴板、鼠标、键盘、窗口前台切换是否正常。

## 命令行参数

| 参数 | 作用 |
|------|------|
| `-Calibrate` | 启动后直接进入校准模式 |
| `-Test` | 发送测试坐标（28.100000, 112.900000）验证输入链路 |

## 技术参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| PointsPerLap | 80 | 每圈采样点数 |
| PaceSecPerKm | 180 | 配速 3:00/km |
| WobbleStdDev | 0.6m | 横向偏移标准差 |
| WobbleSmooth | 0.89 | 偏移平滑度（AR1 系数） |
| PaceVarPct | 0.3% | 配速微变幅度 |


## 工作原理

1. 基于 6 个校准点拟合椭圆跑道（直线段 + 圆弧段），生成 80 点/圈的顺时针路径
2. 通过 AR(1) 过程生成横向摆动序列，叠加到路径上模拟真人跑步的左右偏移
3. 配速加入微变（另一条 AR(1) 序列），使每步间隔不完全均匀
4. 每个点位依次执行：复制纬度 → 点击输入框 → Ctrl+A 全选 → Ctrl+V 粘贴 → 复制经度 → 粘贴 → 点击确认 → 处理弹窗
5. 使用 `keybd_event`/`mouse_event` 旧版 API（不受 UIPI 限制），确保对管理员权限的爱思助手窗口也能正常输入

## 注意事项

- 本工具仅用于学习研究，请遵守相关平台的使用条款
- 校准数据保存在 `data/VirtualRun.cal.json`，窗口大小变化时自动按比例缩放
- 运行期间控制台自动最小化，从任务栏关闭窗口即可停止
