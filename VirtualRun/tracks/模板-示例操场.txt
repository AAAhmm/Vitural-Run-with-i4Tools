# ============================================================
# 自定义跑道坐标模板
# 格式说明:
#   第一行: 跑道名称 + 中文或英文冒号
#   可选:   地址：详细地址
#   坐标行: 方位（纬度,经度）
#
# 必需方位 (6个):
#   右下 / 右上 / 左上 / 左下 / 上顶点 / 下顶点
#
# 坐标用英文逗号或中文逗号分隔均可
# 空行会被忽略, # 开头的行为注释
# ============================================================

示例操场：
地址：湖南省长沙市XX区XX路XX号
右下（28.200000,112.880000）
右上（28.201000,112.880800）
左上（28.201200,112.880200）
左下（28.200200,112.879500）
上顶点（28.201300，112.880500）
下顶点（28.200100，112.879800）
